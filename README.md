# Koolimatemaatika mäng

Autorid: Kaur Kenk, Nansen Palo

Mäng koolimatemaatikast tuntud võrrandite lahendamiseks. Mängus saab lahendada mitut erinevat sorti võrrandit. Mäng on hea võrrandite lahendamise oskuse treenimiseks. \
Vajalik teek programmi käivitamiseks: matplotlib \
Pythoni versioon peab olema vähemalt 3.10 \
Käivitamise juhend: Tuleb teha tühi kaust ja kõik mängu failid sinna sisse panna. Seejärel kas terminalis või koodiredaktoris käivitada __main__.py. \
